# ⚡ FlowIQ

AI-powered onboarding funnel intelligence platform that analyzes user journey data, detects drop-off patterns, and generates actionable product recommendations instantly.

## 🚀 Features

- AI-generated onboarding insights
- Funnel drop-off analysis
- Cohort retention tracking
- Interactive analytics dashboard
- Real-time AI recommendations
- CSV-based event analysis
- Cyberpunk-inspired modern UI

## 🧠 Tech Stack

- Python
- Streamlit
- Plotly
- Pandas
- Groq API
- Llama 3
- LangChain

## 📊 Use Cases

- SaaS onboarding optimization
- Product analytics
- User retention analysis
- Funnel performance monitoring
- Growth experimentation

## 📂 Required CSV Format

```csv
user_id,step,timestamp,completed
U001,Landing Page,2024-05-01 10:00:00,True

## ⚙️ Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔑 Environment Variables

Create `.streamlit/secrets.toml`

```toml
GROQ_API_KEY="your_api_key"
```

## 🌐 Deployment

Deploy easily using Streamlit Community Cloud.

## ✨ Project Highlights

FlowIQ combines AI-powered behavioral analytics with modern product intelligence workflows to help teams identify onboarding friction and improve user retention faster.
